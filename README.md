# ✍️ Handwritten Character Recognition using CNN

CodeAlpha Machine Learning Internship — Task 3  
Recognizing handwritten digits and characters using Deep Learning and Convolutional Neural Networks (CNN).

---

# 📌 Project Overview

This project focuses on building a Handwritten Character Recognition system using Deep Learning techniques.

The model is trained on handwritten image datasets to accurately recognize digits and characters using Convolutional Neural Networks (CNN).

The complete workflow includes:

✔ Data preprocessing  
✔ Image normalization  
✔ CNN model development  
✔ Model training  
✔ Performance evaluation  
✔ Prediction visualization  

The project demonstrates how deep learning can be used for image classification and optical character recognition tasks.

---

# 🎯 Objective

To identify and classify handwritten digits and characters from image data using Deep Learning models.

## ML Problem Type
Image Classification

Examples:
- 0 → Digit Zero
- 1 → Digit One
- A → Alphabet A
- B → Alphabet B

---

# 📊 Dataset Information

## Datasets Used
- MNIST Dataset (Handwritten Digits)
- EMNIST Dataset (Handwritten Characters)

## Dataset Features
- Grayscale handwritten images
- 28 × 28 image resolution
- Multiple digit and character classes

## Target Output
Predicted handwritten digit or character
